3:¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?
R:HTTP:Get, permite realizar peticiones a un recurso especifico a traves de la URL sin enviar datos en el cuerpo de la solicitud.
HTTP:Head, es similar al get pero no regresa ningun contenido en la respuesta, se utiliza para obtener el código de respuesta y el encabezado (header).
HTTP:Post, lo que hace es permitir enviar datos a traves del cuerpo de solicitud sin utilizar el URL tal cual como se hace en el método Get. (Cuando queremos enviar datos al servidor, ejemplo: usuario y contraseña ya que es seguro)
HTTP:Put, es muy similr al metodo Post y puede ser ejecutado multiples veces teniendo el mismo efecto cada vez. (Agregar un recurso)
HTTP:Patch, se utiliza para hacer modificaciones parciales en un recurso especifico, solo actualiza para del código.(Agregar un recurso)
HTTP:Delete, como su nombre lo dice, se utiliza para eliminar un recurso especifico y tambien se puede utilizar multiples veces y teniendo el mismo efecto.

9:¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados.
R:Que un atributo HTML5 este depreciado significa que se considera obsoleto, alguno elementos de HTML 4 que estan desaprobados en HTML 5 son "font", "center", "strike", "applet", "basefont", "frameset" y "frame".

10:¿Cuáles son las diferencias principales entre HTML 4 y HTML5?
R:Las principales diferencias son que HTML5 tiene nuevos elementos semánticos, hay desaprobación de elementos, nuevos atributos, soporta multimedia, funciones nuevas para APIs con fucniones avanzadas de navegador, en general HTML5 ofrece mejor posibilidades de para crear sitios web.

11:¿Qué componentes de estructura y estilo tiene una tabla?
R:Tabla tiene muhcos componentes, "caption" para un título, "colgroup" agrega columnas, "col" aplica atributos a una o varias columnas, "tbody" representa el cuerpo de la tabla, "thead" representa el encabezado de la tabla, "tfoot" es el pie de la tabla, "tr" representa una fila, "td" es una celda regular, "th" es la celda de encabezado.
